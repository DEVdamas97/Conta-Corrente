# Projeto POO com JavaScript

Projeto simples desenvolvido com **HTML**, **CSS** e **JavaScript** com foco em **Programação Orientada a Objetos (POO)**.

## 📌 Objetivo

Praticar os principais conceitos de POO utilizando JavaScript puro no navegador, criando uma aplicação simples e organizada.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🧠 Conceitos aplicados

- Classes
- Objetos
- Métodos
- Encapsulamento
- Organização de código
- Manipulação do DOM
